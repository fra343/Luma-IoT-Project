# Luma-IoT-Project
LUMA è un progetto di Internet of Things (IoT) nato con l'intento di realizzare un'abitazione intelligente capace di gestire in modo automatizzato diverse funzionalità.Il cuore del progetto è la gestione intelligente delle tende, affiancata da un sistema di accesso sicuro tramite tastierino, monitoraggio ambientale e ventilazione.
